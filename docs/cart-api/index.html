    添加商品到购物车

    如果购物车里面已经存在此商品，需要 +1 操作

    • post: cart/add
    // post 参数
{
  "goodsId": 1181008,
  "number": 1,
  "productId": 302
}

// 返回购物车列表

    更新购物车中的某一个商品

    • post: cart/update
    // post 参数
{
  productId: productId,
  goodsId: goodsId,
  number: number,
  id: id
}

// 返回购物车列表

    删除购物车商品

    • post: cart/delete
    // post 参数
{ productId: product_id}

// 返回购物车列表

    勾选购物车商品

    • post: cart/checked
    // post 参数
// productIds: 逗号分隔的id字符串
// isChecked: 0 表示不勾选，1 表示勾选
{
  "productIds": "282,301",
  "isChecked": 0
}

// 返回购物车列表

    选择单个，取消购物车中其他选择项

    • post: cart/exclusiveCheck
    // post 参数
{"productId":302}

// 返回购物车列表
